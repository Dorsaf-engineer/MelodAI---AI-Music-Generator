# 🎵 MelodAI: AI-Powered Music Generation from Text Prompts

MelodAI is a web application that generates original music from natural language descriptions using Artificial Intelligence. Powered by Meta's MusicGen model, the application allows users to enter a text prompt describing the desired music style, mood, or atmosphere and automatically generates a corresponding audio track.

## 🚀 Features

✅ Generate music from text prompts

✅ AI-powered audio generation using MusicGen

✅ Simple and user-friendly web interface

✅ Download generated music as WAV files

✅ Fast and automated music creation

## 🛠️ Technologies Used

| Category | Technologies |
|---|---|
| Backend | Python, Flask |
| AI Model | MusicGen (Meta AI) |
| Machine Learning | PyTorch, Transformers |
| Frontend | HTML, CSS |
| Audio Processing | NumPy, SciPy |

## ⚙️ Installation

### 1. Clone the repository

``` boot
git clone https://github.com/YOUR_USERNAME/MelodAI.git cd MelodAI
```

### 2. Install dependencies

``` boot
pip install -r requirements.txt
```

### 3. Run the application

``` boot
python app.py
```

### 4. Open in your browser
``` boot
http://127.0.0.1:5000
```

## 🤖 AI Model

This project uses **MusicGen**, a state-of-the-art generative model developed by Meta AI for text-to-music generation.

The model converts textual descriptions into audio signals, enabling users to create unique music compositions without any musical expertise.
